## 📖 简介

小石的个人 **Minecraft 模组汉化集合包**，支持多个模组的简体中文翻译。

## 📥 安装方法

1. 前往 [Releases](../../releases) 页面下载最新版本，或点击绿色按钮 **`Code → Download ZIP`**
2. 将 `.zip` 文件放入游戏目录的 `resourcepacks` 文件夹
3. 启动游戏 → 设置 → 资源包 → 启用本汉化包
